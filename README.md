# Stratus ☁️

Stratus is an EVM executor and JSON-RPC server with custom storage that scales horizontally, written in Rust 🦀.

## Current storage implementations

- In Memory
- PostgreSQL

Check our `CONTRIBUTING.md` to get started.

- Just system
- Node.js and NPM
