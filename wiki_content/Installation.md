# Installation Guide

## Prerequisites
- Rust toolchain
- just command runner
- Git
- asdf version manager (for testing)

## Basic Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/cloudwalk/stratus.git
   cd stratus
   ```

2. Install dependencies:
   ```bash
   just setup
   ```

3. Run Stratus:
   ```bash
   RELEASE=1 just run
   ```
